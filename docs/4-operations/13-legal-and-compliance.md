# 13. 법·컴플라이언스

> **면책**: 본 문서는 일반 정보이며 법률 자문이 아니다. 실제 운영 전, 특히 **타인 자산이 연관되거나 제품화할 경우**, 반드시 변호사·세무사의 검토를 받는다. 법령과 규정은 수시로 변한다 — 2026년 현재 기준으로 기술된 내용도 **반드시 최신 확인**이 필요하다.

## 개인용 자동매매의 법적 지위

### 본인 자산 매매 자동화
- **허용**: 본인 계좌에서 본인 자산을 자동 매매하는 것은 금지되지 않음
- 증권사 약관에서 **시스템 트레이딩**을 인정하는 경우가 대부분
- 단, 각 증권사 API 이용약관의 **자동매매 허용 범위** 재확인 필요

### 타인 자산 운용
- **금지 / 등록 필요**: 타인 자산을 운용하면 **투자일임업** 또는 **투자자문업** 등록 대상
- 위반 시 자본시장법상 벌칙 (자본시장법 §17, §445)
- 가족·지인이라도 **수수료를 받거나 운용을 위임받으면** 문제가 될 수 있음

**본 프로젝트 방침**: 본인 계좌에 한정. 타인 자산은 스코프 밖.

## 주요 관련 법령

### 자본시장법 (자본시장과 금융투자업에 관한 법률)
- 투자자문업 · 투자일임업 · 투자매매업 등 규정
- **시세 조종 금지** (§176) — 자동매매로 가장 조심할 영역
- **미공개 정보 이용 금지** (§174) — 내부자 거래
- **부정거래 금지** (§178)

### 개인 자동매매에서 특히 주의
- **허수 주문(spoofing)**: 체결 의도 없는 주문을 반복 → 시세조종 해석 가능
- **자전거래**: 본인 계정 간 매매 (여러 계좌 운영 시 주의)
- **시초가/종가 왜곡**: 동시호가 시간대 대량 주문
- **과도한 빈도의 주문·취소**: 거래소·증권사가 모니터링함

→ 시스템 설계 시 **주문 빈도 상한 · 주문/취소 비율 한도** 도입 (→ [09-risk-management.md](../3-trading/09-risk-management.md))

## 증권사 API 이용약관

### 한국투자증권 KIS 약관 (일반적 조건)
- 본인 계정에서의 프로그램 매매 허용
- **API 남용(초당 호출 초과, 비정상 패턴) 시 차단**
- API 키 3자 양도 금지
- 장애·손실에 대한 면책 조항

### 실전 적용 시
- API 레이트 리밋 엄격 준수
- 장애 발생 시 손실 **자기 책임** — 보험 없음
- 공지사항 정기 확인 (API 스펙 변경 등)

## 로보어드바이저 관련

- **로보어드바이저 테스트베드**(코스콤) — 알고리즘의 안정성·적합성 검증 제도
- 제도적 로보어드바이저 서비스 운영은 **투자일임업 라이선스**가 전제
- 개인용 자동매매는 이 제도 밖

## 세금

### 매매 관련

| 세목 | 대상 | 세율 (2026년 현재 기준, 확인 필요) |
|---|---|---|
| 증권거래세 | 매도 시 | KOSPI 0.03% + 농특세 0.15% = **0.18%**, KOSDAQ 0.18% |
| 양도소득세 (대주주) | 대주주 요건 충족 시 | 20~25% (누진) |
| 배당소득세 | 배당금 | 15.4% 원천징수 |
| 금융투자소득세 | 도입 시점 논쟁 중 | 별도 확인 |

**주의**: 금융투자소득세(금투세) 도입·시행 여부는 정책 변동 — 시행 시 **매매 손익 과세 구조가 바뀜**. 매매 내역 기록은 시행 여부와 무관하게 필수.

### 기록 보관
- 매매 내역·체결가·수수료·세금 **최소 5년 보관** (세무 관련)
- 시스템 로그에 포함하되 **외부 백업** 필수

## 소비자 보호 / 광고 규제

- 본인용이면 해당 없음
- **블로그·SNS에 수익률을 자랑**할 경우:
  - 투자 권유로 해석될 여지
  - 구체적 종목·수익률 공개는 신중
  - "투자 자문이 아니다" 명시

## 지적재산권

- 시세 데이터: **KRX·증권사 약관**에 의해 재배포·상업적 이용 제한
  - 개인 사용은 일반적으로 가능
  - 서비스화 시 반드시 라이선스 확인
- 뉴스 데이터: 언론사·포털 저작권
  - 개인 학습·분석용은 일반적으로 허용 범위
  - 저장·재배포는 제한

## 개인정보보호

- 개인용 시스템이라 외부 수집 개인정보는 없음
- 본인 **계좌번호 · 주민번호 등**은 PII로 취급하여 로그 마스킹
- 클라우드 저장 시 암호화

## 감사 추적성

- 모든 주문·체결은 **append-only 로그**로 보존
- 법·세무 요청 시 제출 가능해야 함
- 시스템이 **왜** 특정 주문을 했는지 **근거(신호 + 전략 버전)** 기록 필수

## 상황별 처리

### "친구에게 같은 시스템 빌려달라"
- 코드 공유 가능 (오픈소스화 여부는 별개 결정)
- **친구 자금을 운용 대신 해주는 것은 금지** — 본인이 직접 자기 계정에서 쓰게 할 것

### "수익을 내니까 제품화하고 싶다"
- **투자자문업 / 투자일임업** 등록 필요
- 자본금 · 인력 · 전산 요건 검토
- 독립적 법률·금융 전문가 상담 필수

### "API 약관이 바뀌었다"
- 최신 약관 리뷰 → 영향 분석 → 시스템 조정 또는 운영 중단
- 약관 위반 시 계좌 정지 위험
